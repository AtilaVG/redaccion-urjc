Con el fin de garantizar una identidad visual coherente en las **II Jornadas Nacionales de Ciencia Abierta** (23 y 24 de marzo, sede de Aranjuez de la URJC), ponemos a tu disposición las plantillas oficiales para la elaboración de tu presentación.

Disponemos de versiones para **[LibreOffice](https://es.libreoffice.org/)** y **Microsoft PowerPoint**, permitiéndote elegir la herramienta que mejor se adapte a tu flujo de trabajo.

### Opciones de descarga

- **Plantilla para LibreOffice:** Utiliza el formato estándar OpenDocument (ODF). Ideal para entornos de software libre.
  - [Descargar plantilla (.odp)](/media/documentos/plantillas/plantilla-jornadas-nacionales-ciencia-abierta.odp)
- **Plantilla para Microsoft PowerPoint:** Formato estándar .potx, compatible con la mayoría de sistemas y equipos.
  - [Descargar plantilla (.potx)](/media/documentos/plantillas/plantilla-jornadas-nacionales-ciencia-abierta.potx)

---

### Instrucciones de uso

Para utilizar la plantilla correctamente, abre el archivo correspondiente y asegúrate de guardar tu trabajo final en el formato adecuado:

1.  **En LibreOffice:** Selecciona `Archivo` > `Guardar como...` > **Presentación ODF (.odp)**.
2.  **En PowerPoint:** Selecciona `Archivo` > `Guardar como...` > **Presentación de PowerPoint (.pptx)**.

> **Nota:** Te agradecemos que utilices estas plantillas en tu intervención para mantener una imagen institucional coherente durante las jornadas.
>
> Para consultar el programa, conocer la organización y tener todos los detalles actualizados, visita la [página web oficial del evento](https://jnca26.gitlab.io/index.html).
